# HelloWorld.Console

Minimal sample for `ControlAgentNet.Channels.Console`.

It combines:

- the base packages from `ControlAgentNet.Agents`
- the `ControlAgentNet.Channels.Console` package
- a tiny inline custom engine to keep the sample credential-free

## Run

```bash
dotnet run --project samples/HelloWorld.Console
```

Type `hello` or any other text. Type `exit` to stop the host.
